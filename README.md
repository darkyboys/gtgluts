# gtgluts
The open source &amp; public domain luts i made for my usage but now available for everyone to use in their projects from this repository

 > Note: The luts hosted here are made by me and are fully Open Source (As their XLUT.GEN-WEB Project is freely available means you can see exactly what color combinations makes these luts and tweak it as per you wish).

And everything here falls in Public Domain.

All the .json files are the source of these LUTs, These needs the [darkyboys/xlut.gen-web](https://github.com/darkyboys/xlut.gen-web) to be opened and modified.
All the .cube files are the exported Ready to use LUTs if you just want to use them without any customizations.

Thank you.
